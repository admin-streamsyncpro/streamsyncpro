Feature 01

Live Chat Control

Monitor incoming TikTok LIVE messages, joins, follows, shares, likes, gifts, treasure boxes, emotes, and viewer activity in a clean desktop feed.

  • Viewer notes, birthdays, profile pictures, first and last activity timestamps.
  • User Points table with ranks, stats, filters, sorting, and manual point adjustments.
Feature 02

Voice And Translation

Read chat aloud, translate comments, moderate abuse, lock voices to viewers, and route TTS through selectable queues so playback stays ordered.

  • TikTok TTS, ElevenLabs with TikTok fallback, and XTTS custom voices with import/export, samples, tuning, cache, rate, and pitch.
  • Moderator commands, blocked words, unsafe prompt filters, cooldowns, slowmode, muted users, and audit logs.
Feature 03

Smart Event Actions

Create custom event actions with a guided builder for follows, likes, comments, shares, joins, gifts, coins, emotes, birthdays, and treasure boxes.

  • Trigger sounds, TikTok TTS, overlays, webhooks, OBS scenes/sources, Voicemod voices, queue media, and cooldown-aware automations.
  • Target everyone, followers, subscribers, moderators, top gifters, specific users, birthday viewers, or selected gifts/emotes.
Feature 04

Overlay Hub And Designer

Manage hosted browser-source overlays from one hub, copy static public URLs, open previews, test overlays, and customise templates visually.

  • Queue, chat, gift, like leaderboard, viewer stats, command feedback, voting, progress bars, Like Race, and Spin Wheel overlays.
  • Drag-and-drop designer with text, images, videos, widgets, voting templates, colours, fonts, animations, reset, import, and export.
Feature 05

Creator Games And Goals

Turn engagement into on-screen moments with voting, progress goals, a wheel of actions, and a TikTok LIVE Like Race mini-game.

  • Like Race uses likes, gifts, avatars, profile pictures, overlays, TTS commentary, countdown sounds, and finish sounds.
  • Progress bars can duplicate, preview, hide background/text, animate goal completion, double/increase goals, or hide when complete.
Feature 06

Music And Integrations

Connect creator tools around the live stream, including Spotify control, OBS scene/source actions, Voicemod voice changes, and webhooks.

  • Spotify sign-in, playlist requests with !play artist song, skip controls, role permissions, and explicit-content filtering.
  • OBS and Voicemod actions can trigger from Event Actions and optionally revert after a configured delay.
