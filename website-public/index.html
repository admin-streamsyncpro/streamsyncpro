Feature 01

Live Chat Control

Monitor incoming TikTok LIVE messages in a clean desktop feed with fast filtering, saved viewer notes, likes totals, and fast status controls.

  • Incoming chat, follows, shares, likes, gifts, and viewer notes in one place.
  • Compact controls designed for live use, not admin spreadsheets.
Feature 02

Voice And Translation

Read chat aloud, translate comments into your selected language, and route TTS through selectable queues so voice playback stays ordered and clear.

  • Built-in, ElevenLabs, and XTTS-style custom voice workflows.
  • Moderation tools, blocked words, cooldowns, voice locks, and cache support.
Feature 03

Smart Event Actions

Create custom event actions for follows, likes, shares, and coins with queue selection, webhook triggers, audience targeting, and sound previews.

  • Trigger sounds, overlays, webhooks, and cooldown-aware automations.
  • Target everyone, subscribers, moderators, specific users, or gift events.
