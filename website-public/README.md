# Stream Sync Pro LIVE Public Website

Upload the contents of this folder to your website root if you want the public marketing page on:

- `https://streamsyncpro.co.uk/`

Suggested upload structure:

- `public_html/index.html`
- `public_html/styles.css`
- `public_html/assets/SSP.png`
- `public_html/downloads/Stream-Sync-Pro-LIVE-Setup-1.0.8.exe`

Notes:

- The main download button now points to a website-hosted installer at `/downloads/Stream-Sync-Pro-LIVE-Setup-1.0.8.exe`.
- Upload or rename the real installer file into `public_html/downloads/` using that exact filename for the button to work.
- If you publish a new version and want the button text/link updated, change `website-public/index.html`.
- Your auth/API files can stay in:
  - `public_html/api/`
  - `public_html/admin/`
